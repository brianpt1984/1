You can download ready-to-use helper files for JSZip demos and related content here.

## Night Pulse project bootstrapper

Download the Windows batch script that provisions the `C:\night-pulse-mtl` directory structure and placeholder files:

- [create_structure.bat]({{ site.baseurl }}/documentation/downloads/night-pulse-structure.bat)

After downloading, double-click the batch file or run it from a command prompt with administrative privileges if required. It will create the folder hierarchy and empty files only when they do not already exist, leaving any existing content untouched.
